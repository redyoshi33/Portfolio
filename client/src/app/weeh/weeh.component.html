Where Eat, Eat Here

GitHub: github.com/redyoshi33/Restaurant_app

Technologies: Python, Django, Google Maps API, Yelp API, Bootstrap

Role: Built on a team of 2 developers, contributed with building the UI, front-end, and back-end.

Description: A restaurant finding web application that allows users to create a profile and add other users as friends. Users can then create groups that use multiple user preferences to select a restaurant.

Awards: Placed 1st in a Python Hackathon amongst 4 teams.
