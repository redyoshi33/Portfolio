24 Game

GitHub: github.com/redyoshi33/24cardgame

Technologies: Express.js, Node.js, Angular, Socket.io, Bootstrap

Role: Worked independently, built the UI, game logic, and server to user connections.

Description: A multiplayer web application game that uses socket.io, in which users compete to see who can make the value 24 using all 4 displayed numbers and the four basic operations of arithmetic (add, subtract, multiply, divide).
