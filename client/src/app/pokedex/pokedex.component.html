Pokedex Trivia

GitHub: github.com/redyoshi33/Pokedex_Trivia

Technologies: Swift, Pokemon API, Core Data

Role: Worked independently, built the UI and the logic, fetching data through API calls.

Description: A trivia iOS application that uses the Pokemon API to pull a description about a random Pokemon and allows the user to guess which Pokemon it is. Users can score points and place on a local score board.

Awards: Placed 1st in a Swift Hackathon amongst 8 teams.
